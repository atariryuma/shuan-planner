# 週案プランナー

小・中学校の先生のための **週案(週指導計画)作成Webアプリ** です。
ブラウザだけで動き、データは端末内に保存されます(サーバー不要・無料)。GitHub Pagesでそのまま公開できます。

## 主な機能

| 機能 | 説明 |
|---|---|
| 📅 週案グリッド | クリックで教科・内容・備考を入力。ドラッグ(PC)/「⇄移動・入替」(タッチ)でコマの入替 |
| 📚 年間指導計画の自動反映 | 教科ごとに単元・時数・各時の内容を登録すると、**週案のコマに単元名と本時の内容が自動で入る**(進度は入力済みコマ数から自動計算)。教科書会社のExcelからの貼り付け取り込み対応 |
| 🔢 時数集計 | 週・年度累計を自動集計。標準授業時数(施行規則別表第一・第二を内蔵、上書き可)との差分・必要ペースを表示。**分数時数(1/3等)・モジュール換算・読み替え合算(書写→国語など)・中止コマ除外**に対応 |
| 🖨 印刷 / PDF | A4横/縦・2種類のレイアウト(縦=校時/縦=曜日)・文字サイズ3段階。**1ページに必ず収まる崩れない印刷**。PDF保存はブラウザの「PDFに保存」でフォント埋め込み不要の検索可能PDFに |
| 🏫 多様な形態 | 小学校/中学校、**学級担任/専科・教科担任(学級ごとの進度自動管理)/複式学級(2学年併記・学年別時数)**、朝のモジュール学習(係数で時数換算)、土曜授業、7校時 |
| 📋 入力省力化 | 基本時間割の登録→毎週ワンクリック反映、前週コピー |
| ☁ Google連携(任意) | Google Apps Script + スプレッドシートで端末間同期、Googleカレンダーから行事欄へ自動取り込み([設定手順](docs/gas-setup.md)) |
| 💾 バックアップ | JSONエクスポート/インポート。PWA対応(ホーム画面に追加してアプリとして利用可) |

## 使い方(最短3ステップ)

1. **設定**タブ: 学校種・学年・担任形態・時程(校時とモジュール枠)を設定
2. **週案**タブ: コマをクリックして教科を選ぶ → 1週間分できたら「その他 ▾ → 基本時間割として登録」
3. **年間指導計画**タブ: 教科書会社の年間指導計画をExcelからコピーして取り込む → 以後、コマに内容が自動反映

毎週は「📋 基本時間割を反映」→ 行事に合わせて微調整 → 「🖨 印刷」だけです。

## GitHub Pages での公開方法

このリポジトリは**ビルド不要**(素のHTML/CSS/JS)です。

1. GitHubに新しいリポジトリを作成(例: `lesson-planner`)
2. このフォルダの中身をすべてプッシュ
   ```bash
   git init
   git add .
   git commit -m "週案プランナー"
   git branch -M main
   git remote add origin https://github.com/<ユーザー名>/lesson-planner.git
   git push -u origin main
   ```
3. リポジトリの **Settings → Pages → Source: Deploy from a branch → Branch: main / (root)** を選択して保存
4. 数分後 `https://<ユーザー名>.github.io/lesson-planner/` で公開されます

> 更新を公開したら [sw.js](sw.js) の `VERSION` を上げてください(PWAキャッシュの更新に必要)。

## 印刷のコツ(重要)

- **Chrome / Edge を推奨**(Safariは用紙の向きをCSSから指定できません)
- 印刷ダイアログで: 用紙 **A4** / 「**背景のグラフィック**」**ON** / 倍率 **100%**
- **PDF保存**: 送信先(プリンター)で「PDFに保存」を選ぶ
- アプリ側の用紙の向き・レイアウト・文字サイズは「🖨 印刷 / PDF」ボタンのダイアログで変更できます

## データの保存について

- データはブラウザの `localStorage` に自動保存されます(端末外には送信されません)
- **長期休業の前には必ず「データ」タブからエクスポート**してください(ブラウザのデータ消去・iPadのSafariの長期未使用時の自動削除に備えて)
- 児童生徒の個人名は入力せず、イニシャル等での運用をおすすめします

## 年間指導計画の取り込み形式

「年間指導計画」タブ → 「📥 取り込み」。次のどちらの表でもOK(Excelからコピー&貼り付け、またはCSV):

```csv
単元名,時数,内容
たし算とひき算,8,筆算のしかた(1)|筆算のしかた(2)|くり上がり|…
かけ算,12,
```

```csv
単元名,内容
たし算とひき算,筆算のしかた(1)
たし算とひき算,筆算のしかた(2)
かけ算,九九の復習
```

テンプレート: [data/年間指導計画テンプレート.csv](data/年間指導計画テンプレート.csv)

> 各教科書会社(東京書籍・光村図書・啓林館・大日本図書・教育出版など)は年間指導計画例をExcel/Word形式で無償公開しています。各社サイトからダウンロードし、単元名・時数の列をコピーして取り込んでください(著作物のため、本アプリには同梱していません)。

## Google連携(任意)

[docs/gas-setup.md](docs/gas-setup.md) の手順で、Google Apps Script(無料)をデプロイすると:

- 複数端末でのデータ同期(職場PC ↔ 自宅PC ↔ iPad)
- Googleカレンダーの予定を週案の行事欄へワンクリック取り込み

が使えます。**設定しなくても全機能オフラインで動作します。**

## 技術構成

- 素のHTML/CSS/JavaScript(ESモジュール)。フレームワーク・ビルドツールなし
- 印刷: `window.print()` + 印刷専用DOM + `@page` 動的注入(mm単位固定レイアウト)
- 保存: localStorage(スキーマバージョン付き)+ JSONバックアップ
- 同期: GAS Webアプリ(text/plain POSTでCORSプリフライト回避)+ スプレッドシート
- PWA: Service Worker(stale-while-revalidate)+ Webマニフェスト
